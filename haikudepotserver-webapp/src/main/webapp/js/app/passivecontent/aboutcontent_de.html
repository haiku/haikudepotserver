Zielsetzung

Diese Anwendung ist ein Online-Tool zur Bearbeitung der Softwarepakete des Betriebssystems Haiku. Die Web-Anwendung bietet dabei die folgenden Funktionen:

  • Einen Online-Katalog aller Pakete.
  • Eine Quelle für zusätzliche Daten für Pakete, die nicht im Standard-Paketformat enthalten sind, beispielsweise:
    • Bildschirmfotos
    • Icons
    • Übersetzungen
    • Bewertungen
  • Eine Benutzeroberfläche zur Bearbeitung der zusätzlichen Daten.
  • Einen Mechanismus um diese zusätzlichen Daten HaikuDepot zur Verfügung zustellen, der Desktop-Anwendung zum Managen von Software-Paketen.
