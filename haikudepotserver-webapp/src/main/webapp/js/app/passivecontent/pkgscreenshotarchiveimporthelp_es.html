Este proceso acepta un archivo tar que ha sido comprimido con gzip. La extensión de archivo para este formato es ".tgz", pero también se suele usar la extensión ".tar.gz". La estructura del archivo comprimido debe seguir la siguiente nomenclatura: "hscr/beshare/1.png". En este ejemplo se proporciona una imagen de mapa de bits PNG como captura de pantalla para el paquete "beshare". Se puede proporcionar calquier cantidad de capturas de pantalla.

El proceso puede operar en dos estrategias. En el modo "Remplazar capturas existentes" se pierden todas las capturas de pantallas y se importan las nuevas. En el modo "Añadir a capturas existentes" se conservan las capturas existentes y las nuevas se adicionan.
