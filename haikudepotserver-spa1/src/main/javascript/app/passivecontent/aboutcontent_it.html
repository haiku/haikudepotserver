Scopo

Questa applicazione web è uno strumento online per lavorare con i pacchetti software del Sistema operativo Haiku (Haiku). L'applicazione web mira a fornire le seguenti funzionalità;

  • Un catalogo dei pacchetti accessibile da Internet.
  • Un repository di dati aggiuntivi relativi a pacchetti non inclusi nello standard formato del pacchetto — come;
    • Screenshot
    • Iconografia
    • Localizzazioni
    • Feedback dell'utente
  • Un'interfaccia utente per manipolare i dati aggiuntivi.
  • Un meccanismo per vendere questi dati aggiuntivi a HaikuDepot, un'applicazione desktop per la gestione dei pacchetti.
