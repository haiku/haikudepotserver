La traducción de paquetes es un respaldo para las traducciones almacenadas en los datos del paquete (versión del paquete). A la izquierda están los diferentes idiomas. Elija uno idioma y se mostrarán sus entradas. Puede editar las entradas, y cuando haya terminado de editarlas para uno o más idiomas, podrá utilizar el botón de acción situado en la parte inferior de la página para guardar los datos. Al momento de visualizar un elemento de traducción determinado (título, resumen o descripción), éste se busca en el siguiente orden de preferencia;

  1. Datos de la versión del paquete para el idioma del usuario
  2. Paquete de datos de repaldo para el idioma del usuario
  3. Datos de la versión del paquete en inglés
  4. Paquete de datos de respaldo en inglés
