La traducción de paquetes es un respaldo para las traducciones que están guardadas en los datos del paquete (versión del paquete). Al lado derecho están los distintos idiomas. Escoja un idioma y serán mostradas sus entradas. Puede editar las entradas y al terminar, puede usar el botón de acción en la parte inferior para guardar los datos. Al momento de mostrar un elemento de traducción (título, resumen o descripción), éste se busca en el siguiente orden de preferencia:

  1. Información de versión del paquete para el idioma del usuario
  2. Respaldo del paquete para el idioma del usuario
  3. Información de versión del paquete para inglés
  4. Respaldo del paquete para inglés
