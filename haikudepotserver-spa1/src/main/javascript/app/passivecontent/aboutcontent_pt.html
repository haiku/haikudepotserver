Fim

Esta aplicação web é uma ferramenta on-line para trabalhar com os pacotes de software do Haiku Operating System (Haiku). A aplicação web visa fornecer a seguinte funcionalidade;

  • Um catálogo acessível à Internet dos pacotes.
  • Um repositório de dados adicionais relacionados com pacotes que não estão incluídos na norma formato pacote — tais como;
    • Screenshots
    • Iconografia
    • Localização
    • Feedback do utilizador
  • Uma interface de utilizador para manipular os dados adicionais.
  • Um mecanismo para vend este dado adicional ao HaikuDepot, uma aplicação de desktop para a gestão de pacotes.
