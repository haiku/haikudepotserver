El repositorio está disponible para varias arquitecturas diferentes. Cada una de ellas está disponible como una "fuente de repositorio" discreta. La fuente de repositorio tiene un número de espejos, uno de los cuales es nominado como el espejo primario y que es donde los datos del repositorio se obtendrán para actualizar los registros en HaikuDepotServer.

Además de los datos reales del repositorio, también hay un archivo repo.info con un campo identifier. Se trata de un identificador único para esa fuente de repositorio real. Este identificador empareja los datos HDS con el repositorio configurado en la aplicación de escritorio HaikuDepot del cliente.
