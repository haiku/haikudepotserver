Le dépôt est disponible pour différentes architectures. Chacune d'elles est disponible en tant que "dépôt source" discret. Le dépôt source a un certain nombre de miroirs, l'un d'entre eux est nommé miroir primaire et c'est de là que les données du dépôt seront obtenues pour rafraîchir les enregistrements du Serveur de dépôt Haiku.

En plus des données réelles du dépôt, se trouve aussi un fichier repo.info avec un champ identifier. Il s’agit d’un identifiant unique pour le vrai dépôt source. Cet identifier associe les données du SDH avec le dépôt configuré dans l’application cliente DépôtHaiku.
