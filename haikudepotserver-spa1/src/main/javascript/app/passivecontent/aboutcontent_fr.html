Objectif

Cette application web est un outil en ligne pour le travail sur les paquets de logiciels du système d’exploitation Haiku (Haiku). L’application web entend fournir les fonctionnalités suivantes :

  • Un catalogue des paquets accessible via internet.
  • Un dépôt de données additionnelles relatives aux paquets non-incluses dans le format de paquet standard — telles que :
    • Captures d’écran
    • Icônes
    • Localisations
    • Retours d’utilisateurs
  • Une interface utilisateur pour manipuler les données additionnelles.
  • Un mécanisme pour distribuer ces données additionnelles à DépôtHaiku, une application de bureau pour gérer les paquets.
