Aquest procés acceptarà un fitxer d'arxiu tar comprimit amb gzip. L'extensió de fitxer típica per a aquest format d'arxiu és ".tgz", però una extensió de ".tar.gz" també s'usa sovint. Els fitxers de l'arxiu haurien de tenir la nomenclatura següent: "hicn/beshare/icon.hvif". En aquest exemple, es proporciona una imatge de mapa de bits PNG com a captura de pantalla per al paquet beshare. Es pot subministrar qualsevol nombre de captures de pantalla com aquesta.

El procés pot funcionar en una de les dues estratègies. Al mode "Reemplaça captures de pantalla existents", es perden totes les captures existents i s'importen les noves. Al mode "Afegeix a les captures de pantalla existents", les captures existents es mantenen i, a més a més, s'importen les noves.
