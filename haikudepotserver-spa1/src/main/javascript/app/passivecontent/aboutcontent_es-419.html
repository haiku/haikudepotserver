Propósito

Esta aplicación web es una herramienta online para trabajar con los paquetes de software del Sistema Operativo Haiku (Haiku). La aplicación web tiene como objetivo proporcionar la siguiente funcionalidad;

  • Un catálogo de los paquetes accesible a través de Internet.
  • Un repositorio de datos adicionales relacionados con los paquetes que no están incluidos en el formato estándar de los paquetes; tales como;
    • Capturas de pantalla
    • Iconografía
    • Traducciones
    • Comentarios de usuarios
  • Una interfaz de usuario para manipular los datos adicionales.
  • Un mecanismo para entregar estos datos adicionales a HaikuDepot, una aplicación de escritorio para la gestión de paquetes.
