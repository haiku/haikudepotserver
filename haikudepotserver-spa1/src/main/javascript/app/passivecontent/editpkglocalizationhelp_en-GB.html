Package localisation is a fall-back for localisation values that are stored in the package data (package version). On the left hand side are the various languages. Choose a language and its entries will be shown. You can edit the entries and when you have finished editing the entries for one or more languages, you are able to use the action button at the bottom of the page to save the data. At display time, a given localisation element (title, summary or description) is chosen in the following order of preference;

  1. Package version data for the user's language
  2. Package fall back data for the user's language
  3. Package version data for English
  4. Package fall back data for English
