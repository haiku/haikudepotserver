Scop

Această aplicație web este o unealtă online pentru lucrul cu pachetele software ale sistemului de operare Haiku (Haiku). Aplicația web țintește să furnizeze următoarea funcționalitate;

  • Un catalog accesibil prin Internet al pachetelor.
  • Un depozit de date adiționale legate de pachete care nu sunt incluse în formatul de pachet standard — precum;
    • Capturi de ecran
    • Iconografie
    • Localizări
    • Feedback utilizator
  • O interfață cu utilizatorul pentru a manipula datele adiționale.
  • Un mecanism pentru a vinde aceste date adiționale la HaikuDepot, o aplicație desktop pentru gestionarea pachetelor.
