Puedes crear íconos de 16px y 32px abriendo un ícono vector en Icon-O-Matic y exportándolo como PNG 16x16px and PNG 32x32px. Se puede obtener el ícono vector abriendo el archivo binario de la aplicación con el ícono usando la extensión para Tracker FileType y haciendo doble clic en el ícono ahí.

El archivo de ícono HVIF se crea de la misma forma, exportando como HVIF desde Icon-O-Matic.

Puedes extraer los íconos 16px y 32px de una aplicación que no tiene un ícono vector (por ejemplo de una antigua aplicación BeOS) usando la herramienta de consola app2png, disponible en HaikuDepot.
