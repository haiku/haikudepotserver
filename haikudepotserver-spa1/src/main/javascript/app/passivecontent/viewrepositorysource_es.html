El repositorio está disponible para varias arquitecturas, cada una de las cuales está disponible como un «repositorio fuente» separado. El repositorio fuente tiene varios espejos, uno de los cuales ha sido designado como el espejo primario y es de donde se obtendrán los datos del repositorio para actualizar los registros en HaikuDepotServer.

Junto con los datos del repositorio también hay un archivo repo.info con un campo identifier. Este es un identificador único de ese repositorio fuente. Este identifier combina los datos de HDS con el repositorio configurado en la aplicación cliente de escritorio HaikuDepot.
