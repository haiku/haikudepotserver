Este proceso aceptará un archivo tar comprimido con gzip. La extensión típica de este formato de archivo es ".tgz", pero a veces también se utiliza la extensión ".tar.gz". Los ficheros del archivo deben tener la siguiente nomenclatura; "hscr/beshare/1.png". En este ejemplo, se suministra una imagen de mapa de bits PNG como captura de pantalla para el paquete "beshare". Se puede suministrar cualquier número de capturas de pantalla.

El proceso puede funcionar con una de dos estrategias. En el modo "Reemplazar capturas de pantalla existentes", se pierden todas las capturas de pantalla existentes y se importan las nuevas. En el modo "Añadir a las capturas de pantalla existentes", las capturas de pantalla existentes se conservan y las nuevas capturas se importan además de las existentes.
