Questo processo accetterà un file di archivio tar compresso con gzip. La tipica estensione per questo formato di archiviazione è ".tgz", ma a volte viene anche utilizzata un'estensione ".tar.gz". I file nell'archivio dovrebbero avere la seguente nomenclatura: "hscr/beshare/1.png". In questo esempio, un file immagine bitmap PNG viene fornito come screenshot per il pacchetto "beshare". Possono essere forniti un numero qualsiasi di tali screenshot.
