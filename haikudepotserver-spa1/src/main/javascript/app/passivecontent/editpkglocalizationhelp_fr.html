La localisation de paquet est un secours pour les valeurs de localisation qui sont stockées dans les données du paquet (version de paquet). Du côté gauche se trouvent les différentes langues. Choisissez une langue et ses entrées seront affichées. Vous pouvez éditer les entrées et lorsque vous avez terminé de le faire pour une ou plusieurs langues, vous êtes en mesure d’utiliser le bouton d’action au bas de la page pour sauvegarder les données. À l’affichage, un élément de localisation donné (titre, sommaire ou description) est choisi suivant cet ordre de préférence;

  1. Données pour cette version du paquet dans la langue de l’utilisateur
  2. Données de repli du paquet dans la langue de l’utilisateur
  3. Données pour cette version du paquet en Anglais
  4. Données de repli du paquet en Anglais
