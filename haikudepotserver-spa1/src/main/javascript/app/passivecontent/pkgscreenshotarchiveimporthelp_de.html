Der Import funktioniert mit einem gzip-komprimierten tar Archiv. Die typische Dateiendung für dieses Format ist ".tgz", jedoch wird auch oft ".tar.gz" benutzt. Die Dateien im Archiv müssen wie folgt benannt werden: "hscr/beshare/1.png". In diesem Beispiel ein PNG Bild als Screenshot für das "beshare" Paket. Es lassen sich beliebig viele solcher Screenshots dazupacken.

Der Import kann auf zwei unterschiedlichen Arten durchgeführt werden:
Im Modus "Vorhandene Screenshots ersetzen" gehen alle existierenden Screenshots verloren und die neuen werden importiert. Im Modus "Zu vorhandenen Screenshots hinzufügen" bleiben die existierenden Screenshots bestehen und die neuen Screenshots werden hinzugefügt.
