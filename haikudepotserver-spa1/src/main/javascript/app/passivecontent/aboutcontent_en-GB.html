Purpose

This web application is an online tool for working with the software packages of the Haiku Operating System (Haiku). The web application aims to provide the following functionality;

  • An internet-accessible catalogue of the packages.
  • A repository of additional data related to packages that is not included in the standard package format — such as;
    • Screenshots
    • Iconography
    • Localisations
    • User feedback
  • A user interface to manipulate the additional data.
  • A mechanism to vend this additional data to HaikuDepot, a desktop application for managing packages.
