Ce processus acceptera un fichier archive tar qui a été compressée avec gzip. L'extension de fichier typique pour ce format d'archive est « .tgz », mais une extension de fichier « .tar.gz » est également parfois utilisée. Les fichiers de l'archive doivent avoir la structure suivante ; "hscr/beshare/1.png". Dans cet exemple, une image PNG est fournie comme capture d'écran pour le paquet « beshare ». Un nombre quelconque de ces captures d'écran peuvent être fournies.

Le processus peut opérer selon deux stratégies. Dans le mode «Remplacer les captures d'écrans exitantes», toutes les captures d'écrans existantes sont perdues et les nouvelles captures d'écran sont importées. Dans le mode «Ajouter aux captures d'écran existantes», les captures d'écrans existantes sont conservées et les nouvelles captures d'écrans sont importées et ajoutées à celles déjà présentes.
