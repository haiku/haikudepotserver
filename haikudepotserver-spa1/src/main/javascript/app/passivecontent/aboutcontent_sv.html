Ändamål

Denna webbapplikation är ett onlineverktyg för att arbeta med programvarupaket för Operativsystemet Haiku (Haiku). Webbapplikationen syftar till att tillhandahålla följande funktioner

  • En internettillgänglig katalog över paketen.
  • Ett arkiv med ytterligare data relaterat till paket som inte ingår i standarden paketformat — Till exempel;
    • Skärmdumpar
    • Ikonografi
    • Lokaliseringar
    • Användarfeedback
  • Ett användargränssnitt för att manipulera ytterligare data.
  • En mekanism för att skicka denna ytterligare data till HaikuDepot, en skrivbordsapplikation för att hantera paket.
