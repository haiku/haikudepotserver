Propòsit

Aquesta aplicació web és una eina en línia per a treballar amb els paquets de programari del sistema operatiu Haiku (Haiku). L’aplicació web té com a objectiu oferir-vos les prestacions següents:

  • un catàleg dels paquets accessible via internet,
  • un dipòsit de dades addicionals relacionades amb els paquets que no s’inclouen en el format estàndard d’empaquetatge, com ara:
    • captures de pantalla,
    • iconografia
    • traduccions, i
    • comentaris dels usuaris
  • una interfície d’usuari per a manipular les dades addicionals, i
  • un mecanisme per a fornir aquestes dades addicionals al HaikuDepot, una aplicació d’escriptori per a gestionar els paquets.
