Назначение

Данное веб-приложение является онлайн инструментом для работы с программными пакетами для операционной системы Haiku (Haiku). Веб-приложение предоставляет следующую функциональность:

  • Доступный через Интернет каталог пакетов.
  • Репозиторий с дополнительной информацией, относящейся к пакетам, но не явяющейся частью стандартного формата пакета — а именно:
    • Скриншоты
    • Иконки
    • Локализация на другие языки
    • Отзывы пользователей
  • Пользовательский интерфейс для управления дополнительной информацией.
  • Механизм передачи дополнительной информации в HaikuDepot, десктопное приложение для управления пакетами.
