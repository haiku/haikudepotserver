Kegunaan

Aplikasi web ini adalah alat online untuk bekerja dengan paket perangkat lunak dari Haiku Operating System (Haiku). Aplikasi web bertujuan untuk menyediakan fungsi berikut:

  • Katalog paket yang dapat diakses internet.
  • Gudang daritambahan data yang berkaitan dengan paket yang tidak terdapat dalam format paket standar — seperti;
    • Scrinsut
    • Icon
    • Lokalisasi
    • Umpan-balik user
  • Antar muka user untuk manipulasi data tambahan.
  • Mekanisme untuk menjual ini data tambahan ke HaikuDepot, aplikasi desktop untuk mengelola paket.
