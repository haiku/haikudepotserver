Paketlokalisering är en reserv för lokaliseringsvärden som lagras i paketdata (paket version). På vänster sida finns de olika språken. Välj ett språk och dess poster kommer att visas. Du kan redigera inläggen och när du är klar med att redigera posterna för ett eller flera språk kan du använda åtgärdsknappen längst ner på sidan för att spara data. Vid visningstidpunkten väljs ett givet lokaliseringselement (titel, sammanfattning eller beskrivning) i det följande preferensordning;

  1. Paketversionsdata för användarens språk
  2. Packets reservdata för användarens språk
  3. Paketversionsdata för engelska
  4. Package reservdata för engelska
